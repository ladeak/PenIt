# PenIt
An application like ZoomIt, but more focused on pen and stylus.

Create a readme that describes this application. Mention that it is inspired by Zoomit.

Usage: 
- CTRL+W zooms into the current position of the cursor.
- Espace to exit from zoomed status.
- CTRL+Pen top button zooms into the current position of the cursor.
- In zoomed position move around by moving the mouse of by moving the pen over the screen.
- To *draw* click with the left mouse button or tap the pen on the screen.
- To switch pen style to yellow highlight hold the side button while tapping the pen on the screen.

This is a WPF application.
